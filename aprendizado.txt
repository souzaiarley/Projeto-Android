1- O uso do article dentro da main mostrou-se uma abordagem interessante.
2- Usar a tag <strong> para colocar algo em destaque.
3- O uso do maior que (>) refere-se ao seletor filho direto.
4- Podemos adicionar variáveis no :root do arquivo css.
5- Usar * para aplicar estilos a todas as caixas html.
6- Usar margin auto para centralizar o conteúdo.
7- Ao usar width: 100%, o conteúdo vai utilizar todo o espaço disponível da caixa.
8- list-style-position: vai ajustar a posição do marcador de uma lista, inside ou outside.
9- columns para determinar em quantas colunas uma lista é dividida.
10- Podemos aplicar valores negativos para margem para expandir uma caixa.
